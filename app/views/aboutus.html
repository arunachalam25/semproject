About Us



The Project is Implementation of an E-Directory Application for Quasi Government using Single Page Application. The E-Directory consist of registered translators details and the language they translate.The project has three types of users admin,user,translator.The user can search for their translators depending on their requirement.The user can also rate the translators,comment on the translators work and also avail online translation tool.The Sentimental analysis is done on the user given comments and graph is plotted.The translator can avail the option to edit their details.The Admin has control to remove,edit,add users and translators to the E-Directory.
